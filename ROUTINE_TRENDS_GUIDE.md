# 📈 成分使用趋势 - 使用指南

## 🎯 功能说明

成分使用趋势页面可以帮助你：
- 📊 查看多天的护肤品使用情况
- 🔍 发现哪些成分你使用最频繁
- ⚠️ 识别可能的成分过度使用
- 💡 获得个性化的护肤建议

---

## 🚀 快速开始

### 步骤 1: 记录每日护肤
1. 点击侧边栏的 📅 **每日护肤** 图标
2. 选择日期
3. 添加晨间/晚间使用的产品和用量
4. 点击 **保存记录**

**示例操作：**
```
选择日期: 2025-12-11
晨间护肤:
  - Paula's Choice BHA 2% → 2 drops
  - CeraVe Moisturizer → 1 pump

晚间护肤:
  - The Ordinary Retinol → 3 drops

[保存记录]
```

### 步骤 2: 连续记录几天
- 建议至少记录 **3-7 天** 的护肤流程
- 每天记录完成后点击保存
- 系统会自动存储到 GCS

### 步骤 3: 查看趋势分析
1. 点击侧边栏的 📈 **使用趋势** 图标
2. 选择时间范围（7天/14天/30天）
3. 查看可视化图表和分析

---

## 📊 页面功能详解

### 1️⃣ 时间范围选择
```
[7天] [14天] [30天]
```
- **7天**: 查看最近一周的使用情况
- **14天**: 两周趋势，更适合观察短期变化
- **30天**: 一个月趋势，适合长期跟踪

### 2️⃣ 每日产品使用量时间线

显示每天使用的产品数量和主要成分：

```
今天                    [2 products]
████████████████████████████ 100%
water 3x  niacinamide 2x  glycerin 1x

昨天                    [3 products]
██████████████████████████████████ 150%
water 4x  retinol 1x  hyaluronic acid 2x
```

**如何解读：**
- **进度条长度**: 相对于这段时间内产品使用最多的一天
- **成分标签**: 当天出现频率最高的前5个成分
- **次数**: 该成分在当天所有产品中出现的次数

### 3️⃣ 成分频率排行榜

显示时间段内最常用的成分：

```
#1  water
    总计: 12x  |  平均: 4.0x
    ████████████████████████████ 100%
    使用于 3天 • ▮▮▮ (迷你时间线)
```

**数据说明：**
- **排名**: #1 表示使用频率最高的成分
- **总计**: 在选定时间段内，该成分出现的总次数
- **平均**: 平均每天使用该成分的次数
- **进度条**: 相对于最高频率成分的百分比
- **迷你时间线**: 每个小柱代表一天，柱子越深表示当天使用量越大

**鼠标悬停查看详情：**
- 每个小柱悬停可看到具体日期和使用次数
- 例如: "2025-12-11: 3x"

### 4️⃣ 成分分析建议

#### 📊 成分分析方法
系统会教你如何解读成分数据：
- **观察高频成分**: 每天都使用的成分是什么？
- **检查重复成分**: 多个产品含相同成分可能过度使用
- **注意活性成分**: 某些成分不宜天天用

#### 🔍 个性化分析
基于你的实际数据生成：
```
• 你最常用的成分是 "water"
• 过去 7 天共使用了 15 种不同成分
• 平均每天使用 2.3 个产品
```

#### ⚠️ 专业建议
常见的护肤注意事项：
- 同时使用多种酸类可能刺激皮肤
- 视黄醇和维生素C建议分开使用（早晚）
- 保湿成分可以每天使用

## 🔍 成分分析实战案例

### 案例 1: 发现成分过度使用

**趋势数据显示：**
```
Salicylic Acid (水杨酸)
总计: 21x | 平均: 3.0x/天
使用于 7天

Glycolic Acid (果酸)
总计: 14x | 平均: 2.0x/天
使用于 7天
```

**分析：**
- ⚠️ 同时使用两种酸类，每天各2-3次
- 💡 建议: 减少频率或隔天使用
- 🎯 优化: 早上用果酸，晚上用水杨酸，或隔天交替

---

### 案例 2: 保湿成分不足

**趋势数据显示：**
```
Retinol (视黄醇)
总计: 7x | 平均: 1.0x/天
使用于 7天

Hyaluronic Acid (透明质酸)
总计: 3x | 平均: 0.4x/天
使用于 7天
```

**分析：**
- ⚠️ 每天使用视黄醇，但保湿成分不足
- 💡 建议: 增加保湿产品频率
- 🎯 优化: 视黄醇前后都使用保湿精华

---

### 案例 3: 合理的护肤流程

**趋势数据显示：**
```
Water (水)
总计: 28x | 平均: 4.0x/天

Hyaluronic Acid
总计: 14x | 平均: 2.0x/天

Niacinamide
总计: 7x | 平均: 1.0x/天

Retinol
总计: 7x | 平均: 1.0x/天
```

**分析：**
- ✅ 基础保湿充足（水、透明质酸）
- ✅ 功效成分适度（烟酰胺、视黄醇）
- ✅ 活性成分控制在每天1次
- 🎯 这是一个平衡的护肤流程

## 💡 高级使用技巧

### 1. 对比不同时间段
```
查看 7天 趋势  → 发现最近一周的变化
查看 30天趋势 → 发现长期使用习惯
```

### 2. 识别成分搭配
观察哪些成分经常在同一天出现：
- 如果 "Retinol" 和 "Vitamin C" 经常同天出现 → 可能需要调整
- 如果 "Hyaluronic Acid" 和 "Retinol" 同天出现 → 合理搭配

### 3. 追踪皮肤状态
结合皮肤状态记录（日历功能）：
- 某天皮肤干燥 → 查看当天成分是否缺少保湿
- 某天皮肤敏感 → 查看当天是否活性成分过多

### 4. 季节性调整
- 夏季: 查看防晒成分使用频率
- 冬季: 查看保湿成分是否充足
- 换季: 对比不同月份的成分变化

## 🎨 可视化图表解读

### 进度条颜色含义

#### 每日产品使用量
- **紫色** (from-purple-500 to-purple-600)
- 代表产品数量的相对比例

#### 成分频率排行
- **绿色** (from-green-500 to-green-600)
- 代表成分使用频率的相对比例

### 迷你时间线
```
▮▮▮▮▮▮▮
```
每个柱子代表一天：
- **柱子高度**: 固定
- **柱子透明度**:
  - 深色 (opacity 1.0) = 使用量最大
  - 浅色 (opacity 0.3) = 使用量最小
  - 中等 (opacity 0.5-0.7) = 中等用量

## ⚙️ 常见问题

### Q1: 为什么显示"暂无数据"？
**A:** 需要先在"每日护肤"页面记录至少一天的护肤流程。

### Q2: 成分统计不准确？
**A:** 确保每次添加产品后都点击"保存记录"按钮。

### Q3: 如何删除错误的记录？
**A:**
1. 进入"每日护肤"页面
2. 选择对应日期
3. 删除错误的产品
4. 重新保存

### Q4: 能否导出数据？
**A:** 当前版本暂不支持导出，数据存储在 GCS，可以通过 API 访问。

### Q5: 迷你时间线看不清？
**A:** 鼠标悬停在每个小柱上可以查看具体日期和使用次数。

## 🔬 成分知识补充

### 活性成分（需控制用量）
- **Retinol/Retinoids** (视黄醇): 建议每晚1次，不超过
- **AHA/BHA** (果酸/水杨酸): 每天1-2次，避免叠加
- **Vitamin C** (维生素C): 每天1次，建议早上
- **Niacinamide** (烟酰胺): 每天1-2次安全

### 保湿成分（可频繁使用）
- **Hyaluronic Acid** (透明质酸): 每天2-3次
- **Glycerin** (甘油): 每天2-3次
- **Ceramides** (神经酰胺): 每天2-3次
- **Squalane** (角鲨烷): 每天2-3次

### 成分搭配建议
✅ **可以同时使用：**
- 烟酰胺 + 透明质酸
- 视黄醇 + 神经酰胺
- 维生素C + 维生素E

❌ **避免同时使用：**
- 视黄醇 + 维生素C (建议早晚分开)
- 视黄醇 + 果酸 (可能过度刺激)
- 多种酸类叠加 (水杨酸+果酸+乳酸)
